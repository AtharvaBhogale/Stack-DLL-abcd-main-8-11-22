Stack Implementation using Doubly Linked List

⊗

To push an element :
1. Create a new node
2. Assign value in the new node
3. if (isEmpty())
{
n->prev = NULL;
n->next = NULL;
// As it is first node
// if stack is empty
start = n;
top = n;
}
else
{
top->next = n;
n->next = NULL;
n->prev = top;
top = n;
}

⊗

To pop an element :
1. Find the top element
2. if (isEmpty())
printf("Stack is empty");
else if (top == start)
{
top = NULL;
start = NULL;
free(n);
}
else
{
top->prev->next = NULL;
top = n->prev;
free(n);
}
